bitwise1

A CD that represents bitwise arithmetic operations on integers. These can be used to represent more specific operations on fixed-width integers, but are intended to be the abstract idealized operation of which fixed-width integer bitwise operations are restrictions. This CD is also supposed to match the idea behind logic1, but is missing two names: true and false. They are missing because false would be 0, and true would be -1, but these are just integers.
