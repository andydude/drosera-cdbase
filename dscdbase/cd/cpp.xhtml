cpp

A CD that represents concepts related to the C Preprocessor (CPP). Since all other CDs represent concepts at the AST level, this cannot represent the C Preprocessor entirely accurately, since it exists at the lexical/token level, and so the points at which these functions may be used is dramatically reduced by forcing macros to only appear in certain locations.
