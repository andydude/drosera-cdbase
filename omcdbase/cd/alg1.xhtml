alg1

A CD of basic algebraic concepts At present this CD only holds definitions of zero and one. They are deliberately defined here without specifying any particular structure (e.g. a group) to which they correspond.

zero a constant
This symbol represents the additive identity element.
Property: for all a | a + 0 = a
∀ a.a+0=a
Property: for all a | 0 * a = 0
∀ a.0⁢a=0
Property: The zero of the integers is 0
∀ x.x∈Z∧x=0⇒x=0
one a constant
This symbol represents the multiplicative identity element.
Property: for all a | 1 * a = a
∀ a.1⁢a=a
Property: for all a | a * 1 = a
∀ a.a⁢1=a
Property: The one of the integers is 1
∀ x.x∈Z∧x=1⇒x=1
