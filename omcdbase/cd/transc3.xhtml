transc3

This CD holds the definitions of many transcendental and related functions. They are defined as multi-valued functions with precise reductions to logs in the case of inverse functions. Note that we use the same names as in the single-valued case, even though it would be traditional to render them with capital letters. In sum <OMS cd="transc3" name="ln"/> is multi-valued, while <OMS cd="transc1" name="ln"/> is single-valued. Note that in many cases A+S only states the log restrictions under some circumstances: JHD has proved (22.8.2002) all the inverse trig. ones

log an application
This symbol represents a binary log function; the first argument is the base, to which the second argument is log'ed. It is defined in Abramowitz and Stegun, Handbook of Mathematical Functions, section 4.1
Property: a^b = c is equivalent to b in Log_a c
log⁡(a,c)={b∈C|ab=c}
Example: log 100 to base 10 (which is {2+2n\pi i}).
log⁡(10,100)
ln an application
This symbol represents the ln function (natural logarithm) as a multivalued function.
Property: y in Ln(x) <=> exp(y)=x
ln⁡(x)={y∈C|exp⁡(y)=x}
Property: y in Ln(x) <=> exp(y)=x
ln⁡(x)={ln⁡(x)+ 2 ⁢n⁢π⁢i|n∈Z}
Example: Ln 1 (which is 0+2n\pi i).
ln⁡(1)
arcsin an application
This symbol represents the arcsin function. This is the multi-valued inverse of the sin function as described in Abramowitz and Stegun, section 4.4. It takes one argument.
Property: y in Arcsin(x) <=> sin(y)=x
arcsin⁡(x)={y∈C|sin⁡(y)=x}
Comment: Only stated in A+S for \z^2|\le 1, but proved for all z in JHD's OpenMath deliverable.
Property: arcsin(z) = -i ln (sqrt(1-z^2)+iz), but the multivalued equivalent is Arcsin(z) = -i Ln (Sqrt(1-z^2)+iz), which we translate into OpenMath as Arcsin(z) = -i [ Ln (sqrt(1-z^2)+iz) union Ln (-sqrt(1-z^2)+iz)],
arcsin⁡(z)={-i⁢y|y∈ln⁡(1-z 2 +i⁢z)∪ln⁡(-1-z 2 +i⁢z)}
arccos an application
This symbol represents the arccos function. This is the multivalued inverse of the cos function.
Property: y in Arccos(x) <=> cos(y)=x
arccos⁡(x)={y∈C|cos⁡(y)=x}
Comment: Only stated in A+S for \z^2|\le 1, but proved for all z in JHD's OpenMath deliverable.
Property: arccos(z) = -i ln(z+i \sqrt(1-z^2)), so the multi-valued equivalent is Arccos(z) = -i Ln(z+i \Sqrt(1-z^2)), encoded as Arccos(z) = -i(ln(z+i \sqrt(1-z^2)) union ln(z-i \sqrt(1-z^2)))
arccos⁡(z)={-i⁢y|y∈ln⁡(i⁢1-z 2 +z)∪ln⁡(i⁢1-z 2 -z)}
arctan an application
This symbol represents the arctan function. This is the multi-valued inverse of the tan function.
Property: y in Arctan(x) <=> tan(y)=x
arctan⁡(x)={y∈C|tan⁡(y)=x}
Property: arctan(z) = (i/2)ln((1-iz)/(1+iz)), so the multi-valued equivalent is Arctan(z) = (i/2)Ln((1-iz)/(1+iz)),
arctan⁡(z)={y⁢i 2 |y∈ln⁡(1-i⁢z1+i⁢z)}
arcsec an application
This symbol represents the multivalued arcsec function as the inverse of sec.
Property: y in Arcsec(x) <=> sec(y)=x
arcsec⁡(x)={y∈C|sec⁡(y)=x}
Property: arcsec(z) = -i ln(1/z+i \sqrt(1-1/z^2)), so the multi-valued equivalent is Arcsec(z) = -i Ln(1/z+i \Sqrt(1-1/z^2)), encoded as Arcsec(z) = -i(ln(1/z+i \sqrt(1-1/z^2)) union ln(1/z-i \sqrt(1-1/z^2)))
arcsec⁡(z)={-i⁢y|y∈ln⁡(i⁢1-1z 2 +1z)∪ln⁡(i⁢1-1z 2 -1z)}
arccsc an application
This symbol represents the multivalued arccsc function as the inverse of csc.
Property: y in Arccsc(x) <=> csc(y)=x
arccsc⁡(x)={y∈C|csc⁡(y)=x}
Property: arccsc(z) = -i ln (sqrt(1-1/z^2)+i/z), but the multivalued equivalent is Arccsc(z) = -i Ln (Sqrt(1-1/z^2)+i/z), which we translate into OpenMath as Arccsc(z) = -i [ Ln (sqrt(1-1/z^2)+i/z) union Ln (-sqrt(1-1/z^2)+i/z),
arccsc⁡(z)={-i⁢y|y∈ln⁡(1-1z 2 +i⁢1z)∪ln⁡(-1-1z 2 +i⁢1z)}
arccot an application
This symbol represents the multi-valued arccot function as the inverse of cot
Property: y in Arccot(x) <=> cot(y)=x
arccot⁡(x)={y∈C|cot⁡(y)=x}
Property: arccot(-z) = - arccot(z)
arccot⁡(-z)={-x|x∈arccot⁡(z)}
Property: arccot(z) = (i/2)ln((1+iz)/(1-iz)), so the multi-valued equivalent is Arccot(z) = (i/2)Ln((1+iz)/(1-iz)),
arccot⁡(z)={y⁢i 2 |y∈ln⁡(1+i⁢z1-i⁢z)}
arcsinh an application
This symbol represents the Arcsinh function as described in Abramowitz and Stegun, section 4.6.
Property: y in Arcsinh(x) <=> sinh(y)=x
arcsinh⁡(x)={y∈C|sinh⁡(y)=x}
Property: Arcsinh z = ln(z +-\sqrt(1+z^2))
arcsinh⁡(z)=ln⁡(z+1+z 2 )∪ln⁡(z-1+z 2 )
Property: Arcsinh(z) = - i * Arcsin(i * z)
arcsinh⁡(z)={-i⁢y|y∈arcsin⁡(i⁢z)}
arccosh an application
This symbol represents the Arccosh function as described in Abramowitz and Stegun, section 4.6.
Property: y in Arccosh(x) <=> cosh(y)=x
arccosh⁡(x)={y∈C|cosh⁡(y)=x}
Property: Arccosh z = ln(z +-\sqrt(z^2-1))
arccosh⁡(z)=ln⁡(z+z 2 -1)∪ln⁡(z-z 2 -1)
Comment: A+S says +/- i ..., but this is irrelevant since Arccos(iz)=-Arccos(iz)
Property: Arccosh(z) = i * Arccos(i * z)
arccosh⁡(z)={i⁢y|y∈arccos⁡(i⁢z)}
arctanh an application
This symbol represents the Arctanh function as described in Abramowitz and Stegun, section 4.6.
Property: y in Arctanh(x) <=> tanh(y)=x
arctanh⁡(x)={y∈C|tanh⁡(y)=x}
Property: Arctanh(z) = - i * Arctan(i * z)
arctanh⁡(z)={-i⁢y|y∈arctan⁡(i⁢z)}
Comment: The condition 0\le x^2 < 1 in A+S is not necessary The proof for Arctan is in JHD's OpenMath deliverable
Property: for all x arctanh(x) = 1/2 * ln((1 + x)/(1 - x))
arctanh⁡(z)={ 1 2 ⁢y|y∈ln⁡(z+11-z)}
arcsech an application
This symbol represents the Arcsech function as described in Abramowitz and Stegun, section 4.6.
Property: y in Arcsech(x) <=> sech(y)=x
arcsech⁡(x)={y∈C|sech⁡(y)=x}
Property: Arcsech z = ln(1/z +-\sqrt(1/z^2-1))
arcsech⁡(z)=ln⁡(1z+1z 2 -1)∪ln⁡(1z-1z 2 -1)
Comment: A+S says +/- i ..., but this is irrelevant since Arcsec(iz)=-Arcsec(iz)
Property: Arcsech(z) = i * Arcsec(i * z)
arcsech⁡(z)={i⁢y|y∈arcsec⁡(i⁢z)}
arccsch an application
This symbol represents the Arccsch function as described in Abramowitz and Stegun, section 4.6.
Property: y in Arccsch(x) <=> csch(y)=x
arccsch⁡(x)={y∈C|csch⁡(y)=x}
Property: Arccsch z = ln(1/z +-\sqrt(1+1/z^2))
arccsch⁡(z)=ln⁡(1z+1+1z 2 )∪ln⁡(1z-1+1z 2 )
Property: Arccsch(z) = i * Arccsc(i * z)
arcsinh⁡(z)={i⁢y|y∈arcsin⁡(i⁢z)}
