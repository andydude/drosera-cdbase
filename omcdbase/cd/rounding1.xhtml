ceiling an application
The round up (to +infinity) operation.
Property: for all x | ceiling(x)-1 < x <= ceiling x
∀ x.(ceiling⁡(x)-1)<x∧x≤ceiling⁡(x)
floor an application
The round down (to -infinity) operation.
Property: for all x | floor(x) <= x < floor(x)+1
∀ x.x<(floor⁡(x)+1)∧floor⁡(x)≤x
trunc an application
The round to zero operation.
Property: for all x | trunc(x) <= x < trunc(x)+1 (x>0)
∀ x.x≥0⇒x<(trunc⁡(x)+1)∧trunc⁡(x)≤x
Property: for all x | trunc(x) >= x > trunc(x)-1 (x<0)
∀ x.x≤0⇒x>(trunc⁡(x)-1)∧trunc⁡(x)≥x
round an application
The round to nearest operation.
Property: for all x | x <= round(x)+1/2 and x <= round(x)-1/2
∀ x.x≤round⁡(x)+1 2 ∧x≥round⁡(x)-1 2
Property: for all x | Also round to even in event of a tie
∀ x.¬(x∈Z)∧x⁢ 2 ∈Z⇒round⁡(x) 2 ∈Z
